# Virtual Cards


##### Activation card for your virtual account

---

### Installation
```
npm install
```

---

### Dependencies

- "express": "4.15.4"
- "graphql": "0.5.0"
- "graphql-tools": "0.3.6"
- "https": "1.0.0"
- "mongodb": "3.0.2"

---

### Usage

```
$ cd virtual\ cards
$ mongodb --dbpath data/db
```

```
$ cd virtual\ cards
$ npm install
$ npm start
```
